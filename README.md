Valentine Page

How to use

- Open `index.html` in a browser (double-click or serve locally).
- Click the `Yes` button to reveal the loving message and hearts.
- Try clicking `No` — it will float away playfully.

Files

- index.html — main page
- styles.css — styling and animations
- script.js — interactive behavior

Notes

- Designed as a deep, loving theme. You can customize colors in `styles.css`.
